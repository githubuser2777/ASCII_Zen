# Contributing to ASCII Zen

First off, thank you for considering contributing to ASCII Zen! It's people like you that make ASCII Zen such a great tool.

## How to Contribute

1.  **Fork the repository.**
2.  **Create a new branch** for your feature or bug fix: `git checkout -b feature/your-feature-name`
3.  **Make your changes.** Ensure your code adheres to the existing style and conventions.
4.  **Test your changes.** If you are adding a new feature, make sure to test it locally.
5.  **Commit your changes** with a clear and descriptive commit message: `git commit -m 'Add some amazing feature'`
6.  **Push to your fork:** `git push origin feature/your-feature-name`
7.  **Submit a Pull Request** against the `main` branch of this repository.

## Reporting Bugs

If you find a bug, please create an issue on GitHub with:
- A clear description of the issue.
- Steps to reproduce it.
- Your operating system and the versions of Go/Python you are using.

## Feature Requests

If you have an idea to improve ASCII Zen, we'd love to hear it! Open an issue on GitHub describing your proposed feature and how it would be useful.
